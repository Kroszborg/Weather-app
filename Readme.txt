# Weather App

The Weather App is a simple web application that allows users to check the current weather and forecast for different locations.

<h2>Features-</h2>

<li>Display current weather conditions (temperature, humidity, wind speed, etc.)</li>
<li>Show weather forecast for Today</li>
<li>Search for weather information by city or location</li>
<li>Responsive design for seamless usage on different devices</li>
<li>User-friendly interface for easy navigation and interaction</li>

<h2>Technologies Used-</h2>

<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>

<h2>Usage-</h2>

<li>Enter the name of the city or location in the search bar</li>
<li>Click the "Search" button to retrieve weather information</li>
<li>View the current weather conditions and forecast for the selected location</li>
<li>Search between different locations to check the weather for multiple areas</li>
